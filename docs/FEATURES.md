# 功能说明

## 短信筛选

### 关键词筛选

- 在短信内容中搜索关键词
- 支持模糊匹配
- 实时搜索

### 号码筛选

- 按号码筛选短信
- 支持部分匹配
- 支持联系人名称匹配

### 正则表达式筛选

- 支持正则表达式语法
- 实时验证正则表达式有效性
- 错误提示

### 日期范围筛选

预设选项：
- 今天
- 最近 7 天
- 最近 30 天
- 最近 90 天
- 自定义范围

### 已读状态筛选

- 全部
- 已读
- 未读

### 锁定状态筛选

- 全部
- 已锁定
- 未锁定

### 消息类型筛选

- 全部
- 收件箱
- 已发送
- 草稿
- 发件箱

### SIM 卡筛选

- 全部
- SIM 1
- SIM 2

### 联系人筛选

- 从通讯录选择联系人
- 按联系人筛选短信

### 筛选条件组合

- 所有筛选条件使用 AND 逻辑组合
- 可以同时应用多个筛选条件
- 筛选结果实时更新

## 批量操作

### 多选模式

- 长按列表项进入多选模式
- 点击列表项切换选中状态
- 显示选中数量

### 全选

- 点击全选按钮选中所有匹配的短信
- 基于数据库查询，不仅仅是当前页
- 显示总选中数量

### 批量删除

- 选择多个短信后批量删除
- 显示删除进度
- 删除完成后显示结果

### 滑动删除

- 左滑列表项显示删除按钮
- 点击删除按钮删除单条短信
- 支持撤销操作

## 导出功能

### CSV 格式

- 采用 RFC 4180 标准
- UTF-8 with BOM 编码
- 包含表头

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| ID | 短信 ID | 12345 |
| 号码 | 发送/接收号码 | 10086 |
| 内容 | 短信内容 | 您的话费余额... |
| 时间 | 发送/接收时间 | 2024-01-01 12:00:00 |
| 类型 | 消息类型 | 收件箱 |
| 已读状态 | 已读/未读 | 已读 |
| 锁定状态 | 锁定/未锁定 | 未锁定 |
| SIM 卡 | SIM 卡标识 | SIM 1 |
| 发送状态 | 发送状态 | 成功 |

### SAF 集成

- 使用 Storage Access Framework
- 用户自定义文件名和保存位置
- 支持覆盖已存在文件

### 进度反馈

- 显示导出进度
- 显示已导出数量
- 显示总数量

## 导入功能

### CSV 解析

- 支持 UTF-8 with BOM
- 支持 RFC 4180 格式
- 自动跳过无效行

### 重复检测

检测规则：
- 号码 + 内容 + 时间三者完全一致才认为是重复
- 跳过重复短信
- 显示跳过数量

### 进度反馈

- 显示导入进度
- 显示已导入数量
- 显示跳过数量

## 默认短信应用管理

### 检测机制

- 启动时检测是否为默认短信应用
- 使用 `Telephony.Sms.getDefaultSmsPackage()`

### 请求机制

**Android 10+ (API 29+)**：
- 使用 `RoleManager`
- 调用 `createRequestRoleIntent(ROLE_SMS)`

**Android 6-9 (API 23-28)**：
- 使用 `Telephony.Sms.Intents.ACTION_CHANGE_DEFAULT`
- 传递包名

### 恢复机制

- 记录原默认应用包名
- 操作完成后显示恢复按钮
- 引导用户前往系统设置

## 筛选历史

### 存储机制

- 使用 SharedPreferences
- 最多保存 5 条历史
- LRU 排序（最近使用的排在前面）

### 操作

- 自动保存筛选关键词
- 显示历史记录
- 快速复用历史记录
- 清空历史记录

## 短信列表

### 分页加载

- 每页 50 条
- 滚动到底部自动加载
- 显示加载状态

### 联系人名称解析

- 从通讯录查询联系人名称
- 显示联系人头像（首字母回退）
- 缓存查询结果

### 关键词高亮

- 在短信内容中高亮显示关键词
- 使用黄色背景
- 支持多个关键词

### 类型标签

- 显示消息类型标签
- 不同类型不同颜色
- 收件箱、已发送、草稿、发件箱

### 可展开内容

- 长短信截断显示（3 行）
- 点击展开按钮显示全部内容
- 支持折叠

## 深色模式

### 主题支持

- Material3 主题
- 跟随系统设置
- 支持动态颜色

### 配色方案

**浅色模式**：
- Primary: #2196F3
- Surface: #FFFFFF
- Background: #F5F5F5
- Error: #F44336

**深色模式**：
- Primary: #90CAF9
- Surface: #1E1E1E
- Background: #121212
- Error: #EF9A9A
